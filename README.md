# Packing List Generator

A dynamic travel packing list generator built with **HTML, CSS, JavaScript, Node.js, and Express**.
Users answer a series of travel-related questions, and the application generates a personalized packing checklist based on their selections.
Live on:https://packinglist-mb4n.onrender.com/

## Features

* Multi-step questionnaire
* Next/Previous navigation
* Form validation
* Travel-specific recommendations
* Categorized checklist
## Technologies Used

### Frontend

* HTML5
* CSS3
* JavaScript (Vanilla JS)

### Backend

* Node.js
* Express.js


## Packing List Categories

Generated packing items are organized into:

* Personal Items
* Electronics
* Utilities
* Health
* Clothes
* Accessories
* Swimwear
* Footwear
